Deep Learning dengan Python

Website ini dibuat untuk membantu teman-teman yang masih kesulitan dengan bahasa Inggris agar tetap bisa mempelajari konsep deep learning dengan lebih mudah dan bertahap.

Penulis buku asli: François Chollet. Buku aslinya bisa dilihat di https://deeplearningwithpython.io/. Hak cipta konten asli tetap milik penulis dan penerbit terkait.

Untuk François Chollet: saya ucapkan banyak terimakasih. Saya membaca buku ini sewaktu di akhir perkuliahan sebelum menyusun tugas akhir penelitian, saat kesulitan menemukan referensi Deep Learning yang komprehensif dan gratis. Buku ini kemudian menjadi pondasi belajar saya dalam pembelajaran Deep Learning. Karena saya belum terlalu memahami bahasa Inggris, saya cukup kesulitan membacanya, sehingga repository ini saya buat sebagai catatan pribadi agar lebih mudah dipelajari kembali di waktu lain.

Visi Proyek

Menyederhanakan akses ilmu deep learning bagi pembaca Indonesia yang belum menguasai bahasa Inggris teknis, tanpa menghilangkan makna konsep aslinya.

Format Konten

Setiap bab difokuskan pada terjemahan isi ke Bahasa Indonesia dengan gaya bahasa sangat sederhana agar pembaca pemula lebih cepat paham.
